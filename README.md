# Portafolio personal docente — Juan Carlos Blandón Vargas

Sitio web tipo hoja de vida/portafolio profesional, desarrollado con HTML, CSS y JavaScript vanilla, optimizado para GitHub Pages.

## Estructura

```text
/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── img/
│   ├── icons/
│   └── fonts/
└── README.md
```

## Publicación en GitHub Pages

1. Sube este proyecto a un repositorio en GitHub.
2. Ve a **Settings > Pages**.
3. En **Build and deployment**, selecciona:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (o la rama principal) y carpeta `/root`
4. Guarda los cambios y espera la URL pública.

## Personalización rápida

- Reemplaza los placeholders en `assets/img/` por evidencias reales.
- Actualiza correo, LinkedIn y demás enlaces de contacto en `index.html`.
- Ajusta métricas y textos según resultados verificables.
